Installation

Before you install Hit the Road theme, please make sure you have latest version of Ghost installed on your blog. The latest Ghost version that Hit the Road was tested against is 1.25.5

The list of Ghost blog releases can be find here.

Upload through admin dashboard

Log into the dashboard of your blog and navigate to Settings » Design.

Once there, click the Upload theme button. A file input dialog will appear. Select theme zip file that you have downloaded from your Themeforest account. After the theme is uploaded a new dialog will appear, asking if you want to activate the theme right now or just close the dialog. Click the Activate now button.

Search Widget

To get the search working, either the old Public API or the new Content API must be enabled.

First, create a new post and set it's title to "Search" (without quotes). Then, open the Post Settings and select the "Turn this post into page" checkbox. Also, the template options must be changed to "Search".

Public API, pre Ghost v2.10

The search widget will appear automatically if the Public API is enabled.

The Public API can be enabled on the Labs page.

Content API, post Ghost v2.10

First, you have to obtain an access key from the admin dashboard.

While on the admin dashboard, go to Integrations page and click the Add custom integration button. Enter "Theme integration" as integration name and click the Create button. On the next page, select and copy the Content API key.

Now, replace content_api_key in the following code with the actual Content API key from the Integrations page.

<meta name="content-api:access-key" content="content_api_key">

After that, navigate to Code injection page and copy that code into Site header field and click the Save button.

Instagram Widget

The Instagram Widget can show photos from your Instagram account on the sidebar of your blog. To get it working, the Instagram User ID and Access Token must be obtained first. Here is a short guide how to get Instagram User ID and Access Token, if you have never done it before.

Once you have it, go to the Code injection page in the administration area of your blog. Paste the following code into the Blog Header text field:

<meta name="instagram:user-id" content="{user-id}" />
<meta name="instagram:access-token" content="{access-token}" />

Replace {user-id} with your Instagram User ID and {access-token} with actual Access Token you have obtained from your Instagram account.

Save the changes you have made by clicking the Save button in the upper right corner of the page.

Comments

Ghost blog does not have its own comment system yet. But the theme has built-it support for Disqus commenting platform.

To turn on Disqus comments, first, you have to register for Disqus account here, or log into your account if you already have it and create a new site. Just remember the shortname of your newly created site, you will need it in the next step.

Then, on your blog dashboard, navigate to the Code injection page and copy the following code into the Blog Header field:

<script>
    var disqus_shortname = '{disqus-shortname}';
</script>

Replace {disqus-shortname} with the value of your site's shortname and click the Save button on the upper right corner of the page.

Weather Widget

Lets say you're writing an article about how to spent a nice weekend in L.A. and you'd like to display short weather forecast, so the reader could get an idea what the weather will be at this coming weekend. So you just simple put an extra meta tag after your article, e.g.:

<meta name="weather-widget:location" content="Los Angeles, CA" />

And it's done! The widget will appear in the sidebar next to your post. Actually, for your convenience this meta tag can be placed anywhere in the post.

Setup

You can configure the weather widget in Code Injection page on your Ghost blog. If Blog Header textbox is empty, paste following code into it:

<meta name="weather-widget:provider" content="yahoo" />
<meta name="weather-widget:provider-key" content="" />
<meta name="weather-widget:loading-title" content="Loading weather data..." />
<meta name="weather-widget:units" content="c" />

Provider

Currently the widget supports two weather forecast providers.

Yahoo Weather

You can obtain required credentials for accessing Yahoo Weather API by registering your site on Yahoo Developer Network.

<meta name="weather-widget:provider" content="yahoo" />
<meta name="weather-widget:provider-client-id" content="yahoo_client_id" />
<meta name="weather-widget:provider-key" content="yahoo_client_secret" />
<meta name="weather-widget:provider-app-id" content="yahoo_app_id" />

Open Weather Map

Open Weather Map has a free plan with 50 000 requests per day and permits their weather data to be used with non-commercial and commercial purposes. Set provider to Open Weather Map by injecting the following code to the blog header:

<meta name="weather-widget:provider" content="openweathermap" />

You have to acquire an API key and inject it to the blog header too:

<meta name="weather-widget:provider-key" content="<place_your_api_key_here>" />

Units

What units should be used to display weather temperature. C for celsius and F for fahrenheit.

<meta name="weather-widget:units" content="c" />

How to use

At the end of your article you can specify the location for the weather widget by adding this meta tag:

<meta name="weather-widget:location" content="San Francisco, CA" />

And that's it! The weather widget will appear in the sidebar along your article.

Subscribers

The Subscribers is a feature of Ghost blog that allows you to capture email addresses on your blog, which can then be exported and imported.

The Subscribers feature is not enabled by default, so in order to use it on your blog you have to enable it first.

Inside your blog dashboard, go to Settings » Labs, and check the Subscribers box under the Beta features part of the page. After that, the Subscribers widget will appear on your blog.

Read more about Subscribers feature here.

Maps Widget

Before using Google Maps widget in your blog posts, you have to obtain Google Maps API key. If you never done it before, here is a link to a quick guide how to do that.

Once you have the API key, use following example to put map widget in your blog posts:

<div class="widget widget-inline render-component-location"
    data-api-key="your Google Maps API"
    data-title="Redwood National and State Parks"
    data-lat="41.2131788" data-lng="-124.0046276"
    data-zoom="10"
    data-phone="+1 707-465-7335"
    data-location="California, United States"
    data-website="http://nps.gov"
    data-hours="Open 24 hours"
    data-show-marker="true"
    data-desc="Immense woodland system containing many towering redwood trees, with trails, campgrounds & more."></div>

As you can see, the widget can be configured by setting data attributes on it's HTML tag.

The API key, latitude and longitude properties are mandatory. If no other properties are present, map will expand to fill it's container horizontally.

Google Analytics

First of all, you have to obtain the Google Analytics tracking code. If you don't know how to get the tracking code, here is the help article from Google that explains how to do that.

Once you have the tracking code, go to the Code Injection page in your blog dashboard, and insert the code into Blog Footer field. Remember to save the changes you have made by clicking the Save button on the upper right corner of the page.

Navigation

Links to the navigation bar can be added through the admin panel or by editing navigation.hbs template.

Drop-down menu

A second level menu can be added to the navigation bar. To do this, open the navigation.hbs template and locate the ul tag. New menu items must be added inside the ul tag.

Below is the example of HTML snippet that creates a second level menu that lists all the islands of Canaries archipelago.

<li class="dropdown">
    <a href="#" class="dropdown-toggle" data-toggle="dropdown">Canaries<b class="caret"></b></a>
    <ul class="dropdown-menu">
        <li><a href="#">El Hierro</a></li>
        <li><a href="#">Fuerteventura</a></li>
        <li><a href="#">Gran Canaria</a></li>
        <li><a href="#">La Gomera</a></li>
        <li><a href="#">Lanzarote</a></li>
        <li><a href="#">La Palma</a></li>
        <li><a href="#">Tenerife</a></li>
    </ul>
</li>

Dropdown menu with Map Glyphs

Various cartographic elements can be displayed using Map Glyphs web font by MapGlyps.com. Because of licensing issues, the Map Glyphs font is not bundled with the theme. But you can download it from MapGlyps.com and use it on your blog for free.

Download the font from their site and extract everything into assets/components/mapglyphs folder.

If a dropdown menu contains map glyphs items, in addition to dropdown-menu class it must contain dropdown-menu-destinations class too, e.g.:

<ul class="dropdown-menu dropdown-menu-destinations">

Below is the example of dropdown menu item that has a map of Europe next to it when viewed on the blog.

<li><a href="{{@blog.url}}tag/europe"><i class="mg map-wrld-eu"></i>Europe</a></li>

The list of all available elements can be found here.
